# Blog Posts
Welcome to the news blog which informs about the learning process.
This section of the repository can be used for writing and publishing blogposts about the project.

Write a blog post in a separate Markdown document in the folder 'blog'.
Save the file. Then add a link to the blogpost below.
Use the following formatting:
```
- [Second Post - Date - Topic - Author](/blog/filename_of_second_post.md)
- [First Post - Date - Topic - Author](/blog/filename_of_first_post.md)
```
